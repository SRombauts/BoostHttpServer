Welcome to the Boost C++ HTTP Server

This is a very simple C++ embeddable web server build from the Boost.Asio multithreaded HTTP 1.0 Server Example. It is released under the Boost license (see LICENSE_1_0.txt) that encourages both commercial and non-commercial use.

Boost provides free peer-reviewed portable C++ source libraries that work well with the C++ Standard Library. Boost libraries are intended to be widely useful, and usable across a broad spectrum of applications.

HTML source code Doxygen documentation is available in the doc/html/ directory

Compatibility: Boost 1.35 (introduction of Boost.Asio)
Tested under the following development environnement :

  • Ubuntu Natty (11.04)/Debian Squeeze (6.0) with gcc 4.x
  • Windows XP/Seven with Visual Studio Express 2008
